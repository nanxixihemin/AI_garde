## 部署指南 (针对您的云服务器)

无论您使用的是阿里云、腾讯云、华为云，还是海外 VPS，均可选择以下两种方式之一进行秒级部署：

### 方法一：使用 Docker Compose 部署 (强烈推荐 ⭐⭐⭐⭐⭐)

此方式免去了在服务器安装 Node.js、npm 和配置进程守护的麻烦，且数据自动在宿主机持久化。

#### 1. 克隆项目到服务器
SSH 登录您的服务器，在您希望存放项目的目录下运行：
```bash
git clone https://github.com/nanxixihemin/AI_garde.git
cd AI_garde
```

#### 2. 一键启动容器
```bash
docker compose up -d --build
```
*   **注**：此命令会在后台构建并运行容器。默认映射在服务器的 **`3000`** 端口（您可以通过编辑 `docker-compose.yml` 中的 `"3000:3000"` 来更改，如修改为 `"80:3000"` 即可直接通过 HTTP 80 端口访问）。
*   **数据安全**：容器运行后，系统会自动在服务器当前目录下创建 `data/` 文件夹。您填写的成绩、打卡等所有私人数据都会实时保存在宿主机的 `data/` 下，重启容器或更新代码不会丢失任何数据。

---

### 方法二：使用 Node.js + PM2 部署 (传统方式)

如果您不想使用 Docker，可以直接在服务器上通过 Node.js 运行：

#### 1. 安装依赖
确保服务器已安装 Node.js (推荐 v18 或以上)。
```bash
git clone https://github.com/nanxixihemin/AI_garde.git
cd AI_garde
npm install --production
```

#### 2. 使用 PM2 守护进程运行 (防止终端关闭后服务挂掉)
如果您没有安装 `pm2`，可以全局安装它：
```bash
npm install -g pm2
```
启动服务：
```bash
pm2 start server.js --name "ai-evolution"
```
查看运行状态：
```bash
pm2 status
```
开机自启：
```bash
pm2 save
pm2 startup
```

##  端口放行与域名绑定

1.  **安全组放行**：请登录您的云服务器后台（如阿里云控制台/腾讯云控制台），在“安全组”或“防火墙”设置中，放行 **`3000`** 端口（入站规则，协议 TCP，源地址 `0.0.0.0/0`）。
2.  **手机填报与访问**：放行后，您就可以直接在手机浏览器或电脑浏览器中输入 `http://您的服务器公网IP:3000` 进行数据查看与填报了！
3.  **反向代理与域名 (可选)**：如果您有域名，可以使用 **Nginx** 配置反向代理，并申请 Let's Encrypt 免费 SSL 证书实现 HTTPS 访问：
    ```nginx
    server {
        listen 80;
        server_name yourdomain.com; # 替换成您的域名

        location / {
            proxy_pass http://127.0.0.1:3000;
            proxy_set_header Host $host;
            proxy_set_header X-Real-IP $remote_addr;
            proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        }
    }
    ```
